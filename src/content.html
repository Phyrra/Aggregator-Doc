Installation
The library is registered as a npm package. To use it, simply execute yarn add sama-aggregator You can find the code on Github.
Usage
The library is written in Typescript, but can be used as a node module in JavaScript.

To use in browser, there is a script generated by browserify, located in dist/masa-aggregator.browser.js. The Aggregator is registered on the window object for global access.

(Any): Returns a found element, or null

Arguments

-or-

condition (Function): A condition to be evaluated

value (Any): A value to compare against
-or-
mapper (Function): A function to map the element for comparison

-or-
mapper (Function): A function to map the element for comparison
value (Any): A value to compare against

Checks if a value matches the provided reference, based on the following rules
  • The search term will be split on space
  • Any non-word characters will be stripped
  • parts that are too short will be ignored
  • Matches need not be exact, everything within a certain distance will be included
Returns
Boolean: true if the value is a complete or partial match, false if not
match( [( String | Function ), ] String )
Arguments
key (String): A property key, may be composited with . (e.g. state.id)
reference (String): A value to compare against
-or-
mapper (Function): A function to map the element for comparison
reference (String): A value to compare against
Example
a.where( match('description', 'word') ); a.where( match('description', 'multiple words') ); a.where( match('description', 'mispelled') );
